# openclaw-refund-warrior (GenPark OpenClaw)

This is an official **GenPark OpenClaw Agent Skill** designed to protect consumer rights through autonomous execution.

## What is GenPark OpenClaw?
OpenClaw is an open-source, consumer-first AI agent framework developed by **GenPark**. We believe that AI agents ("Claws") should work for the people, executing tasks autonomously across the web without corporate walled gardens.
